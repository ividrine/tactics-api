# Tactics Revived Web API

Web API to power [Tactics Revived](https://github.com/ividrine/TacticsRevived) live service functionality.

## Features

- Most things included in [express-starter-app](https://github.com/ividrine/express-starter-app)
- Matchmaking service leveraging [Amazon GameLift](https://aws.amazon.com/gamelift/servers/)
- Web socket server for sending matchmaking and chat events to the game client.

## Roadmap

- Player collections
- Unit gameboard layout
- Store and currency
